Chapter 5

間斷隨機變數及其機率分配

隨機變數把隨機實驗的結果轉成數值。當可能值是有限或可數時，就屬於間斷隨機變數。本章介紹機率分配、期望值、變異數，以及二項、超幾何與波松分配。

學習目標

  1. 理解隨機變數與機率質量函數。
  2. 計算間斷隨機變數的期望值與變異數。
  3. 辨認伯努力試驗、二項分配、超幾何分配與波松分配。
  4. 分辨抽出放回與抽出不放回對分配的影響。

本章地圖

本章先說明如何把事件結果數值化，再討論機率如何分配在各可能值上。接著介紹三個常見模型：二項分配處理固定次數的成敗試驗，超幾何分配處理有限母體不放回抽樣，波松分配處理事件發生次數。

核心內容

5-1 隨機變數的意義與種類

隨機變數是把樣本點對應到實數的函數。例如丟 3 次硬幣，若 X 表示正面次數，X 可能是 0、1、2、3。每個可能值對應的機率形成機率分配。

5-2 間斷隨機變數的機率分配

間斷隨機變數的機率以機率質量函數表示。所有可能值的機率必須大於等於 0，且總和等於 1。期望值是長期平均結果，變異數衡量結果相對期望值的波動。

5-3 二項式分配

若試驗次數固定、每次只有成功或失敗、成功機率固定且各次試驗獨立，成功次數 X 服從二項分配。投籃命中次數、抽驗瑕疵品數都常用此模型。

5-4 超幾何分配與波松分配

超幾何分配處理有限母體中抽出不放回的成功次數。波松分配描述單位時間或空間內事件發生次數，例如每週車禍次數、每頁錯字數、每 10 分鐘提款人數。

5-5 機率質量函數與累積分配函數

間斷隨機變數的分配可以用表格或函數列出。例如 X 表示一天內銷售的電視機台數，可能值為 0、1、2、3、4，若每個值都有對應機率，這張表就是機率分配。判斷是否為合法分配時，要檢查每個機率都介於 0 與 1 之間，且全部機率加總為 1。

累積分配函數 F(x)=P(X≤x) 表示 X 不超過某個值的機率。它會隨 x 增加而不下降，且可由機率質量函數逐步累加而來。若要由累積分配函數找出單點機率，可用相鄰累積機率相減，例如 P(X=3)=F(3)-F(2)。

5-6 期望值、變異數與線性轉換

期望值不是一次實驗必然出現的結果，而是長期重複實驗後的平均位置。若 X 的可能值為 x，機率為 f(x)，則 E(X)=Σxf(x)。變異數衡量各可能值離期望值的平均平方距離，標準差則把單位拉回原變數單位，較容易解讀。

若新的變數由 Y=aX+b 形成，例如銷售台數 X 轉成營收或利潤，則 E(Y)=aE(X)+b，Var(Y)=a²Var(X)。這個性質常用在把隨機變數轉成金額、分數或成本時。

5-7 常見間斷分配的判斷

伯努力分配描述單次成敗試驗；二項分配描述固定 n 次獨立伯努力試驗中的成功次數。若母體有限且抽出後不放回，成功機率會隨抽樣改變，此時應使用超幾何分配，而不是二項分配。

波松分配適合描述固定時間、面積或空間中事件發生次數，前提是事件平均發生率穩定，且不同區間的發生情況可視為獨立。當二項分配的 n 很大、p 很小，且 np 適中時，也常用波松分配近似二項分配。

名詞解釋

5-1 隨機變數

  • 隨機變數 Random variable：把隨機實驗結果轉成數值的函數。
  • 間斷隨機變數 Discrete random variable：可能值為有限或可數的隨機變數。

5-2 機率分配

  • 機率質量函數 Probability mass function：列出每個可能值及其機率的函數。
  • 期望值 Expected value：隨機變數的長期平均。
  • 變異數 Variance：隨機變數相對期望值的平均平方偏離。

5-3 二項式分配

  • 伯努力試驗 Bernoulli trial：只有成功與失敗兩種結果的單次試驗。
  • 二項分配 Binomial distribution：固定 n 次獨立伯努力試驗中的成功次數分配。

5-4 超幾何分配與波松分配

  • 超幾何分配 Hypergeometric distribution：有限母體不放回抽樣的成功次數分配。
  • 有限母體校正因子 Finite population correction factor：不放回抽樣時修正變異數的因子。
  • 波松分配 Poisson distribution：描述固定區間內事件發生次數的分配。

5-5 分配函數

  • 累積分配函數 Cumulative distribution function：表示 P(X≤x) 的函數。
  • 合法機率分配 Valid probability distribution：所有機率非負且總和為 1 的分配。

5-6 分配特徵

  • 線性轉換 Linear transformation：把 X 轉成 aX+b 的變數轉換。
  • 標準差 Standard deviation：變異數的平方根，單位與原變數相同。

5-7 分配選擇

  • 放回抽樣 Sampling with replacement：每次抽出後放回，成功機率可視為固定。
  • 不放回抽樣 Sampling without replacement：每次抽出後不放回，成功機率會隨抽樣改變。
  • 波松近似 Poisson approximation：在 n 大、p 小時用波松分配近似二項分配。

公式說明

E(X) = Σ x f(x)，Var(X) = E(X²) - [E(X)]²
二項分配：P(X=x)=C(n,x)p^x(1-p)^(n-x)
若 X~Binomial(n,p)，E(X)=np，Var(X)=np(1-p)
波松分配：P(X=x)=e^-λ λ^x / x!，E(X)=Var(X)=λ

範例解析

範例 1：二項分配

某選手命中率 0.65，投 20 球，命中 15 球的機率為 C(20,15)(0.65)^15(0.35)^5。此情境符合固定次數、成功機率固定與獨立試驗。

範例 2：超幾何分配

8 位候選人中有 3 男 5 女，選 3 人且不放回。選到 2 男 1 女的機率為 C(3,2)C(5,1)/C(8,3)。

範例 3：波松分配

某提款機 10 分鐘平均 4 人提款，10 分鐘內剛好 5 人提款的機率為 e^-4 4^5 / 5!。若改成 30 分鐘，λ 要同步放大為 12。

自我測驗

  • 能確認隨機變數的可能值。
  • 能由機率質量函數計算期望值。
  • 能辨認二項、超幾何與波松分配情境。
  • 能正確調整波松分配的時間或空間區間。

重點整理

間斷隨機變數把不確定結果轉成可計算的數值。期望值描述長期平均，變異數描述波動。二項、超幾何與波松分配對應不同資料產生機制，使用前必須先確認試驗條件。
