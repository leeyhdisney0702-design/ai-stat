課程入口

統計學互動式 HTML 教材

本教材以統計學課程的學習順序編排，提供適合課堂投影與學生自學的互動式網頁。每章會包含學習目標、核心內容、專有名詞解釋、公式說明、範例解析、練習題、答案與重點整理。

目前已完成：第 1 章導論、第 2 章統計圖表。後續章節會依同一套教材格式擴充。

章節清單

  • 第 1 章 導論

    認識統計學、統計流程、敘述統計與推論統計、母體與樣本。

  • 第 2 章 統計圖表

    學習資料型態、衡量尺度、資料蒐集方法與圖表選擇。

  • 第 3 章 統計測量數

    待製作。

  • 第 4 章 機率論

    待製作。

使用說明

  1. 正式版本可直接透過 GitHub Pages 網址開啟，不需要安裝 Python 或啟動本機伺服器。
  2. 本機開發時可用任一靜態檔案伺服器預覽，但頁面連結與資源路徑皆使用相對路徑。
  3. 頁面左側有導覽目錄，可快速跳到指定區塊；右上方可顯示或隱藏目錄。
  4. 練習題答案預設隱藏，點擊「顯示答案」即可檢查。
  5. 互動題可即時回饋，適合課堂提問或學生自我檢核。

建議上課順序

先從第 1 章建立統計推論的基本語言，再進入第 2 章學會辨識資料類型與選擇圖表。確認樣板後，再擴充第 3 章統計測量數與後續推論章節。
